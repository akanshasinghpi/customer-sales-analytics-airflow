# Customer Sales Analytics Pipeline (Airflow + Python + SQL)

## Overview
Built an end-to-end data pipeline to process sales data, generate insights, and automate workflows using Apache Airflow.

## Tech Stack
- Python (Pandas)
- SQL (SQLite)
- Apache Airflow
- Docker
- Matplotlib

## Pipeline Architecture
Raw Data → ETL → Database → Analytics → Reporting → Visualization → Airflow Automation

## Features
- Modular ETL pipeline (Extract, Transform, Load)
- Automated workflow using Airflow DAG
- SQL-based analytics layer
- Automated report generation
- Visualization of business insights

## Key Learnings
- Built ETL pipelines using Python
- Orchestrated workflows using Airflow
- Managed containerized environments using Docker
- Handled file paths in container environments

## How to Run
1. Run Docker Airflow
2. Trigger DAG in Airflow UI
3. Outputs generated in `/output`
